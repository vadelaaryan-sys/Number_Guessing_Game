A simple and interactive Number Guessing Game built using HTML, CSS, and JavaScript.

## 📌 About the Project
This game automatically generates a random number in the background.  
The user enters a number and the game checks whether the guessed number is correct.

## 🚀 Features
- Random number is generated automatically
- User enters a number as a guess
- Displays **"Congratulations! You are Right"** when the guessed number matches
- Displays **"Too High! Try Again"** if the entered number is greater than the generated number
- Displays **"Too Low! Please Try Again"** if the entered number is smaller than the generated number
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎮 How It Works
1. Open the game in the browser
2. Enter a number in the input field
3. Click the check button
4. The game compares your input with the generated random number
5. The result message appears instantly
